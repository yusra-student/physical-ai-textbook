# Module 5: Advanced Intelligent Systems

Welcome to the Advanced Systems module! In this module, we will explore how to integrate Large Language Models (LLMs), secure authentication, and intelligent task execution into our robotic systems.

## What you will learn

- **RAG Brain**: How to give your robot a brain that can read and answer questions from documentation.
- **Secure Agent**: How to protect your robot's API using Role-Based Access Control (RBAC).
- **Intelligent Tasks**: How to build reusable, resilient task workflows that can adapt to failure.

## Prerequisites

- Completion of Modules 1-4
- Basic understanding of Python and Docker
